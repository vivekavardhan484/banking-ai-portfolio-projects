# Banking AI Portfolio Projects

This repository contains machine learning projects focused on solving real-world banking and business problems using Python and data science techniques.

## Projects Included

### 1. Bank Fraud Detection System
Built a machine learning classification model to identify potentially fraudulent banking transactions.

Key areas covered:
- Data preprocessing
- Handling imbalanced datasets
- Logistic Regression
- Random Forest
- Model evaluation using precision, recall, F1-score, and ROC-AUC
- Confusion matrix and ROC curve visualisation

Business relevance:
Fraud detection is critical in banking because missed fraudulent transactions can lead to financial loss and customer trust issues.

---

### 2. Customer Churn Prediction
Developed a machine learning model to predict whether customers are likely to leave a financial service provider.

Key areas covered:
- Exploratory data analysis
- Feature engineering
- Classification modelling
- Feature importance analysis
- Customer behaviour prediction

Business relevance:
Customer churn prediction helps businesses improve customer retention and reduce long-term revenue loss.

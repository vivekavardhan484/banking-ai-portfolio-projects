# Banking AI Portfolio Projects

This repository pack contains two GitHub-ready projects for AI/Data roles in banking and consulting:

1. Bank Fraud Detection System
2. Customer Churn Prediction

Each project includes:
- Python code
- README file
- Requirements file
- Business problem explanation
- Model evaluation
- Saved model output
- Visualisation output

These projects are designed to support applications for roles such as:
- Data Analyst
- Junior Data Scientist
- AI Graduate Consultant
- Technology Analyst
- Risk Analytics Graduate
